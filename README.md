# Сайт Сімейного центру ментального благополуччя «Твоя опора»

Статичний сайт (один файл `index.html` + папка `assets/`). Без бекенду й бази — тому **нікому не треба давати доступ до адмінки**. Усі правки — у тексті `index.html`.

## Що викладати на хостинг

Достатньо двох речей:
- `index.html`
- папку `assets/` (логотип, ластівка, фото гостей)

**Не потрібні** для роботи сайту (це лише для локального перегляду): `server.js`, `README.md`.

Розмістити в корені субдомену `center.tvoya-opora.org` (субдомен основного сайту `tvoya-opora.org`). Сайт одразу працює.

## Як додати подію в «Анонси подій»

1. У файлі `index.html` знайдіть `<!-- ЯК ДОДАТИ ПОДІЮ -->`.
2. Скопіюйте один блок `<article class="ev-card reveal">…</article>`.
3. Змініть:
   - дату — `<span class="date">14<small>липня</small></span>`
   - аудиторію — `<span class="pill">Підлітки</span>` (Дорослі / Усі / Діти…)
   - колір шапки — `style="background:var(--violet)"` (можна `--coral`, `--gold`)
   - назву `<h3>…</h3>` та опис `<p>…</p>`
4. **Форма реєстрації (Google Форма):**
   - Створіть Google Форму під подію → кнопка **«Надіслати» → іконка посилання → копіювати**.
   - Вставте це посилання в кнопку:
     `<a class="btn btn-primary" href="ТУТ_ПОСИЛАННЯ_ФОРМИ" target="_blank" rel="noopener">Зареєструватися</a>`
   - Поки форми немає — лишіть `href="#booking"` (веде на загальну форму запису).

> Кожна подія = окрема Google Форма. Відповіді збираються у вашу Google-таблицю, доступ до сайту для цього не потрібен.

## Як додати/змінити гостя «Давай на ти»

Знайдіть `<!-- Щоб додати гостя -->`. Скопіюйте блок `<div class="guest reveal">…</div>`:
- з фото: `<div class="ph"><img src="assets/файл.png" alt="Імʼя"></div>` (файл покладіть у `assets/`)
- без фото (ініціали): `<div class="ph">ІП</div>`
- підпис: `<div class="gname">Імʼя</div>` та `<div class="grole">Роль · дата</div>`

## Як змінити контакти / години роботи

Шукайте в `index.html`: телефон `111 18 00`, години `Пн–Пт 9:00–18:00`, адресу `Ворзельська, 24`, Telegram/Instagram — і правте текст напряму.
